Введение
========

.. hint::

   Для лучшего понимания устройства модулей расширения, предварительно ознакомьтесь с
   :doc:`руководством по управлению модулями</manage/site/plugins/index>`.

Модули расширения, как следует из названия, расширяют возможности Eresus CMS. Они могут делать это
следующими способами:

#. :doc:`предоставлять новый тип раздела<guide/providing_content_type>`;
#. :doc:`обрабатывать события<guide/events>`;
#. :doc:`добавлять пункт(ы) в меню «Расширения»<guide/add_item_to_extensions_menu>`.
